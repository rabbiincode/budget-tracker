# Budget Tracker Application

A brief description of features of this application

# Key Features

- Add Income and Expense transactions.

- Select categories for each transaction.

- Specify date and amount for accurate tracking.

- Optional description for additional details.

- Snackbar notifications when a transaction is added.

- Responsive UI using MUI Grid and components.

- Form validation: all mandatory fields must be filled before submitting.


# Design Choices

- TypeScript: Ensures type safety and reduces runtime errors.

- Material-UI: For a clean, responsive, and professional UI.

- Context API: Handles global state for transactions efficiently.

- Local Storage: To persist data.

- Validation & Disabled Button: Prevents adding incomplete transactions.

- Description Field Optional: Allows users to add notes without forcing input.


# Improvements with Time

- Add recurring transactions and monthly summaries.

- Improve UI with custom themes, icons, and animations.

- Implement voice input or smart suggestions based on transaction description.


Time Spent

Approximately 6-8 hours building the app, including designing the form, implementing state management, and integrating MUI components.
